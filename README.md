# Online store Moire

This online store has catalog with filters, product detail page. User can add product to cart and place an order.


## Setup 
```bash
# npm
npm install
```

## Development Server

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```

## Stack
* Vue 3
* Typescript
* Vitest
* SCSS
* API Swagger
